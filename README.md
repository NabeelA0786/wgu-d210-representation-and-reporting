# D210 – Representation and Reporting

**WGU Master of Science in Data Analytics**

## Overview
Designed and built an interactive Tableau dashboard combining telecom churn data with U.S. Census income data to communicate business insights to a non-technical audience.

## Dataset
- Telecom customer churn dataset
- ACS 5-Year Income Estimates by ZIP code (U.S. Census Bureau, 2023)

## What Was Done
- Merged churn data with geographic income data by ZIP code
- Built an interactive Tableau dashboard presenting KPIs and trends
- Designed visualizations for executive-level stakeholder communication
- Focused on clarity, layout, and actionable insight delivery

## Tools
`Tableau` · `Python` · `Pandas`
